Se muestra un formulario, el cual hay que rellenar sin dejar vacío ningun campo
y tecleando la misma contraseña en ambos campos.

En caso de no introducir nombre, mostrará aviso debajo del campo Nombre 
que no puede estar vacío.

En caso de no introducir correctamente las contraseñas (han de ser iguales) o 
de que estén vacías mostrará mensaje de error guardando en caso de haber
indicado un usuario, el nombre de este.

Se ha realizado mediante CSS una hoja de estilo para encuadrar el formulario en el centro.
Se ha añadido un fondo de pantalla y modificado algunas propiedades de aspecto en
textos, campos y parrafos.
